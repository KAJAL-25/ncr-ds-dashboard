# NCR DS Dashboard

This project is an interactive Streamlit dashboard that analyzes
Data Scientist job opportunities in the NCR region.

## Features
- Filters by Location, Work Mode and Role
- KPI cards for company count and salary insights
- Skill distribution using pie chart
- Company rating analysis using bar chart
- Package vs Rating analysis
- Top companies by package

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Excel

## How to Run
pip install -r requirements.txt
streamlit run app.py
